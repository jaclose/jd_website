# JD-1184 — working notes

Personal site of Jafar Dabbagh. Solar-system-as-navigation hero; sections are
celestial bodies (see README for the map).

## Architecture invariants

- The hero canvas is **fixed, full-viewport, pointer-events: none**. R3F
  raycasts from events bubbling on `#page-root` (`eventSource` +
  `eventPrefix="client"`). Never give the canvas wrapper pointer events.
- The 3D scene and DOM overlays communicate through
  `components/hero/store.ts` — positions are written per-frame into a plain
  module object; only `hovered` goes through React state. Keep setState out
  of frame loops.
- Scroll progress drives the genie collapse: each body lerps along a
  quadratic bezier from orbit to a screen-space dock slot. The system docks
  into a **floating centered pill** (`NavBar.tsx`); the DOM pill and the
  scene must agree through `slotCenters`/`sunSlotX`/`pillBounds`/`BAR_TOP`/
  `BAR_H` in the store. Slot anchor y is `BAR_TOP + 24`.
- Homepage sections are full-viewport scenes (`min-h-svh`, full-bleed) paged
  by mandatory lenis snap (`SectionSnap`, desktop pointers only) — keep each
  scene's content inside one viewport height on desktop.
- The garden planet texture is generated from `biosphere()` in
  `data/garden.ts` — change growth stages there, never hardcode colors.
- Content JSON in `content/` is generated by `scripts/import_wordpress.py`;
  don't hand-edit migrated essays.

## Style

- Palette/typography live in `app/globals.css` (`@theme`): space black,
  warm ink, starlight gold, hairlines at 12% white. Fraunces (display),
  Cormorant Garamond (essay body), IBM Plex Mono (micro-labels, `.label`).
- Mono labels are uppercase with 0.28em tracking. Hairline borders
  everywhere, no heavy strokes, no rounded corners except planets and the
  docked nav pill.
- Use Tailwind v4 canonical class forms (`text-[8px]!`, `bg-linear-to-t`,
  `aspect-3/4`, `border-hairline`, quarter-rem spacing) — the Problems pane
  stays at zero.

## Verify

running server captures hero / docked bar / garden for visual review.
